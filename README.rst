.. start-intro

TensorFlow MRI is a collection of TensorFlow add-ons for computational MRI.

.. end-intro

Installation
------------

.. start-install

You can install TensorFlow MRI with ``pip``:

.. code-block:: console

    $ pip install tensorflow-mri

Note that only Linux wheels are currently being provided.

.. end-install
