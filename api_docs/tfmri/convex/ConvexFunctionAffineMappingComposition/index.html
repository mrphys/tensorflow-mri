tfmri.convex.ConvexFunctionAffineMappingComposition#

class ConvexFunctionAffineMappingComposition(function, operator=None, constant=None, scale=None, name=None)[source]#

Bases: tensorflow_mri.python.ops.convex_ops.ConvexFunction

Composes a convex function and an affine mapping.

Represents \(f(Ax + b)\), where \(f\) is a ConvexFunction, \(A\) is a LinearOperator and \(b\) is a constant Tensor.

Parameters

  • function – A ConvexFunction.

  • operator – A LinearOperator. Defaults to the identity.

  • constant – A Tensor. Defaults to 0.

  • scale – A float. A scaling factor.

  • name – A name for this operation.
